## YAML Snippet Shape

```yaml
- symbol: LITE
  status: watching
  added: 2026-04-29
  thesis_stop_mcap_b: <user-owned>
  target_mcap_b: <user-owned>
  thesis_summary: |
    <user-owned thesis text>
  themes: [theme_ai_compute, theme_optical_module]
```

Allowed status values for current system docs are `active`, `watching`, and `removed`. Only `active` is scored.
