## 架构调整重点

### 1. 双池 SoT 走 yaml

- a 池 SoT = `config/a_pool.yaml`（不是 Notion / 不是 DB）
- m 池 override = `config/m_pool_overrides.yaml`
- themes SoT = `config/themes.yaml`（V5.7 新增 · ADR-028）
- DB 镜像 3 字段：pool / is_active / thesis_added_at

### 2. 价格锁 → 市值锁

- a_pool.yaml：`thesis_stop_mcap_b` / `target_mcap_b`（十亿 USD）
- 价格反推：`mcap * 1e9 / shares_outstanding`
- 拆股增发免疫

### 3. shares_outstanding NOT NULL · NULL hold + alert

- ETL 每周拉取 · 存 `symbol_universe.shares_outstanding`
- NULL → 个股信号走 hold · alert_log WARN

### 4. themes_score_daily 现产现耗

- 不向后填
- 某日某主题缺分 → 全部依赖该主题的 a 股信号走 hold

### 5. A 池 verdict_text 骨架兑底

- LLM 失败 / GCP 超限 / Vertex key 过期 都不能让 daily 崩
- 骨架：【代码】A_Score=X · top_signal · 入场 Y / 止损 Z
